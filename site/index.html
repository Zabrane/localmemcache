Efficiently sharing a hashtable between processes on a local Unix machine.

Localmemcache aims to be faster than using memcached locally by using shared memory and providing a similar interface for Ruby and C.

Install

The Ruby binding is available as a Ruby Gem. It can be installed by executing

gem install localmemcache

Using

require 'localmemcache'
$lm = LocalMemCache.new :namespace => :viewcounters
$lm[:foo] = 1
$lm[:foo]
$lm.delete(:foo)

Read more.

Performance

Here's a quick speed comparison, made on an Intel(R) Xeon(R) CPU E5205 @ 1.86GHz:

Benchmark pseudo code: 
2_000_000.times {
  index = rand(10000).to_s
  $hash.set(index, index)
  $hash.get(index)
}

MemCache:      253,326.122 ms
LocalMemCache: 6,055.552 ms
Ruby's Hash:   4,963.313 ms
Localmemcache is about 40 times faster than using memcached locally, and about 20% slower than Ruby's hash.
